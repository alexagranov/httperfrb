HTTPERF.rb

Simple Ruby interface for httperf.

Requires httperf, of course...

Installing 'httperf'

Mac

sudo port install httperf

Debian / Ubuntu

sudo apt-get install httperf

Redhat / CentOS

sudo yum install httperf
